History of the Internet


Early World Wide Web

On March 12, 1989 the British computer scientist, Tim Berners-Lee proposed a more effective means of communication at the European Organization for Nuclear Research (CERN). Eventually, Berners-Lee realized that this system can also be used throughout the world. His colleague and computer scientist Robert Cailliau suggested to use hypertext links to access various kinds of information as “a web of nodes in which the user can browse at wil” which lead to Berners-Lee producing the very first website in 1991.

In essence the World Wide Web is an information space where documents and other web resources are identified by URIs, interlinked by hypertext links, and can be accessed via the Internet which is now more commonly known as simply the web. The web consists of webpages that contain hypertext documents. These hypertext document are merely text documents formatted and annotated with the Hypertext Markup Language (HTML). Later on, webpages developed to contain links to images, video, and software components that are rendered to users of a web browser application. Furthermore, when multiple web pages are published within a common domain name, the collection is called a website.

Once the web west up, several browsers started to get develop come 1993. Mosaic was considered as the ‘web's transition to the mainstream’ because of its appearance that is user-friendly. Come 1994, Netscape Navigator followed which attained an 80% share of web browser usage by 1996.

Later on in 1996, the Internet Explorer 3.o was launched by Microsoft which was considered a risk for the web to become a proprietary technology because most of its advanced features (like Media Player) can only work for IE. Which means that non-IE user cannot be able to use the apps.

Other websites such as Yahoo! made an impact in the web history for its searchable directories of websites apart from it being also a search engine. However, when Google entered the race, it proved to have a more focused technology and indexing that made it a gateway to the whole web.
